# Nilesoft Shell Script Sublime Text Syntax

A Sublime Text syntax definition for Nilesoft Shell Script files.

## Installation

Install using Sublime Text's Package Control:

1. Open Sublime Text and open the Command Palette using `Cmd+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac).
2. Type `Package Control: Install Package` and select the package from the list.
3. Search for `Nilesoft Shell Script` and select the package to install.
